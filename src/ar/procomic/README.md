# ProComic - Tachiyomi Extension

مصدر ProComic لتطبيق Tachiyomi - توسيع لقراءة المانجا والمانهوا والمانهوا والروايات الرسوم الخاصة على منصة ProComic.

## المميزات ✨

**البحث والاستعراض:**
- البحث عن المانجا والمانهوا والروايات
- التصفح حسب النوع (مانجا، مانهوا، مانها، رواية)
- قوائم التحديثات الأخيرة

**القراءة:**
- عرض الفصول بشكل منظم
- دعم الصور من خوادم CDN
- معلومات المترجمين والتاريخ

**المزايا الإضافية:**
- دعم المصادقة عبر الـ Token
- تخزين بيانات المستخدم محليا

## المتطلبات

- Android 5.0+ (API 21)
- Tachiyomi 2.0+
- اتصال بالإنترنت

## التثبيت

1. قم بتحميل ملف `.apk` المجمع
2. ثبت التطبيق على جهازك
3. افتح Tachiyomi وأضف المصدر من قائمة المصادر
4. ابدأ في البحث والقراءة
