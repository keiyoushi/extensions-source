## FAQ

### What is this extension?
This is an implementation for the current Bato v4 sites (bato.si, bato.ing).

### Why is there another extension for Bato.to?
The official Bato.to extension targets the v2 site and mirrors. This one is for v4 and is kept separate to avoid breaking existing users.

### Bato.to (v4) is not loading anything?
Bato.si can be blocked by some ISPs. Try a VPN or a different network.
